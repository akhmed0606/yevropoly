Warranty

Yevro Polycarbonate provides written warranty documentation for eligible polycarbonate sheets supplied in Canada. Our warranties are UV Performance warranties (not general “everything” warranties) and apply only when products are stored, handled, installed, and maintained correctly.

If you need the written warranty documents, download them below:

Multiwall Polycarbonate

UV Performance Warranty — 10 Years

Eligible Yevro-supplied multiwall polycarbonate sheets include a written 10-year UV Performance warranty.

This warranty covers:
UV-related performance (UV protection performance), as defined by the warranty terms

Key conditions (summary):

  • Store sheets properly prior to installation (flat, supported, protected from heat build-up and weather exposure)
  • Install correctly and allow for thermal expansion/contraction
  • Install the UV-protected side correctly where applicable
  • Use compatible tapes, sealants, and cleaners
  • Follow proper end-closure practices for multiwall sheets where applicable
Download: Download Multiwall UV Warranty (10 Years) — PDF

Solid Polycarbonate

UV Performance Warranty — 15 Years

Eligible Yevro-supplied solid polycarbonate sheets include a written 15-year UV Performance warranty.

This warranty covers:
UV-related performance (UV protection performance), as defined by the warranty terms

Key conditions (summary):

  • Store sheets properly prior to installation (flat, supported, protected from heat build-up and weather exposure)
  • Install correctly and allow for thermal expansion/contraction
  • Use compatible sealants and cleaners (avoid harsh solvents)
  • Avoid exposure to incompatible chemicals
Download: Download Solid UV Warranty (15 Years) — PDF

Important notes (please read)

These are UV Performance warranties only. They do not cover physical damage, scratches, impact damage, chemical damage from incompatible products, or issues caused by incorrect storage, handling, fabrication, or installation.

Labor, removal/re-installation, project delays, and consequential damages are not covered, except where required by law.

Warranty eligibility and remedies are governed by the written warranty terms in the PDF documents.
